# c-Shop Api

1. Clone proyect
2. ```Yarn install```
3. Clone file ```.env.template``` and rename to ```.env```
4. Change environment variables.
5. Up database.
```
docker-compose up -d
```
6. Run Seed
```
http://localhost:3000/api/v2/seed
```
7. run proyect ``` yarn start:dev ```
